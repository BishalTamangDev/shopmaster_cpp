### ShopMaster - Shop Management
"ShopMaster" is a simple yet powerful system designed to streamline day-to-day shop operations. Built in C++ programming language, it provides a complete solution for managing inventory, generating bills, tracking transactions, and analyzing sales reports.

#### Features
1. Inventory Management
   - Add Product
   - View Products
     - All
     - Available
     - Out of Stock
   - Update Product
   - Restock Product
   - Remove Product
   - Search Product [Using ID]
2. Report & Searching
   - View All Sale
   - View Today's Sales Report
   - Search Sales Report
     - Annual
     - Monthly
     - Daily
     - Using Customer Name
   - Search Invoices 
3. User Management
   - Admin
     - Add
     - Update Details
     - Update Password
   - Employee
     - Add
     - Update Details
     - Update Password
     - Remove Employee
4. Customer Management
   - View customers
   - Search Customer using Sales ID
   - Remove Customer using Saled ID
